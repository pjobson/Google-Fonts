# Google Font Archive

Archive of (Google Fonts)[https://fonts.google.com/]

https://github.com/jrnewell/goog-webfont-dl

## Setup

Install nodejs, I use `tj/n`, you can use whatever you want.

    curl -L https://bit.ly/n-install | bash

Install Dependencies

    npm install
    apt install wget
